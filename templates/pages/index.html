What is OpenBotX

A structured AI orchestration platform built to manage, coordinate, and monitor multiple intelligent agents in a secure and organized way — designed from the ground up with production environments in mind.

Security-First Architecture

Each agent runs inside an isolated workspace with sandbox restrictions, limiting file access and tool permissions. Controlled and predictable by design.

Simple Installation

No complex infrastructure, no broken dependencies, no heavy configuration. A simple installation process designed to work without headaches.

Real-Time Web Dashboard

A complete production-ready dashboard to manage agents, monitor tasks, track logs, and control executions — accessible from desktop, tablet, and mobile.

Agent Orchestrator

Not just another AI agent — a structured orchestrator that manages multiple agents, each capable of spawning subagents with tiered tool access and permission isolation.

Key Capabilities

Production-ready features built for real-world deployment.

Real-Time Dashboard

See everything happening in real time — manage agents, monitor tasks, control executions, track logs, and inspect error traces.

Kanban-Style Board

Track in real time what each agent is doing with a visual Kanban board — structured task management with full transparency.

Sandboxed Workspaces

Each agent operates inside an isolated workspace with sandbox restrictions and tiered tool permissions, preventing uncontrolled behavior.

Real-Time Skill Editing

Add and edit skills directly through the web dashboard — no restarts, no manual file editing, no environment conflicts.

Multi-Channel Support

Built-in web chat and native Telegram support with text, audio, and images — using the same processing pipeline.

Hierarchical Agents

Agents can spawn and manage subagents with tiered access levels — a scalable and structured architecture for complex workflows.

Integrated File Browser

Native file browser inside the web panel with public and private file separation — all operations within sandboxed environments.

Multi-Provider LLM

Support for Anthropic, OpenAI, and Google services with flexible model configuration and prompt caching for reduced latency.

Independent Web Chat

Built-in web chat that does not depend on external applications — supports text, audio, and images with full communication control.

Agent-to-Agent Routing

Intelligent message routing between agents with contextual continuity — enabling structured collaboration between specialized LLM instances.

Visual Config Editor

Visual configuration editing with YAML validation — making advanced configuration safer and easier without breaking the system.

Memory & Continuity

Persistent memory and session handling that keeps context across interactions — providing continuity across conversations and tasks.

Built-in Tools

A native Python tools system that transforms OpenBotX into a programmable AI automation platform.

File Operations

Sandboxed read, write, and file management with public and private separation — safe access without system exposure.

Shell Execution

Secure shell access with safety protections and controlled permissions for running system commands.

HTTP Requests

Full HTTP support including GET, POST, and custom headers — enabling direct integration with external APIs and services.

Web Search & Fetch

Search the web and fetch content directly from agents — bringing real-time external information into workflows.

Browser Automation

Chrome/CDP integration for controlled interaction with web environments — secure browser automation built into the platform.

Image Generation

Generate images directly from agent conversations — integrated visual content creation as part of the workflow.

RSS Feeds

Monitor and consume RSS feeds as part of automated workflows — keeping agents updated with external content sources.

Scheduling & Cron

Cron-based task scheduling for recurring automations — set up processes that run on your schedule without manual intervention.
